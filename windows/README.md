# JC Budgeting for Windows

This folder contains the Windows client and standalone server downloads.

## What Is Here

- `x64/manual/desktop/`: portable desktop client files
- `x64/manual/server/`: portable standalone server files
- `x64/installers/`: Windows installer `.exe` files

## Recommended Install Method

- Use the files in `x64/installers/` for the easiest setup
- Run `JCBudgeting-Client-Setup-<version>.exe` to install the desktop client
- Run `JCBudgeting-Server-Setup-<version>.exe` to install the standalone server

## Manual Use

- The client can be started from `x64/manual/desktop/JCBudgeting.exe`
- The standalone server can be started from `x64/manual/server/Start JCBudgeting Server.cmd`
- The Windows server tray companion is included in the manual server folder and is started by the server launcher

## Notes

- The server page is available at `http://localhost:5099/server` after the server starts
- Use the installers when possible for easier install and uninstall behavior
